Реализация игры Тетрис на Python с использованием библиотеки Pygame. Код адаптирован для работы как на десктопе (клавиатура), так и на мобильных устройствах (сенсорные кнопки).

Установка и запуск

bash:
pip install pygame
python tetris.py
